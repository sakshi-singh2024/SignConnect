# SignConnect ✋🔊

🚀 Real-time Sign Language Detection using Computer Vision

## 📌 Overview
SignConnect is a real-time hand gesture recognition system that converts sign language into text and speech using computer vision.

## ✨ Features
- ✋ Hand gesture detection using webcam
- 🔤 Alphabet recognition (A–Z)
- 💬 Word detection (HELLO, YES, NO, APPLE, THANK YOU)
- 🔊 Text-to-speech output
- 🖥 Interactive GUI using Tkinter

## 🧠 Tech Stack
- Python
- OpenCV
- MediaPipe
- Tkinter
- pyttsx3

## ⚙️ How It Works
1. Captures live video using webcam
2. Uses MediaPipe to detect hand landmarks
3. Applies rule-based logic to identify gestures
4. Converts gestures into text
5. Speaks output using TTS engine
