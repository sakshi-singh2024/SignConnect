# SignConnect ✋🔊

## Overview
SignConnect is a real-time hand gesture recognition system that converts sign language into text and speech.

## Features
- Detects hand gestures using webcam
- Converts gestures into alphabets (A–Z)
- Supports basic words like HELLO, YES, NO
- Text-to-speech output
- GUI using Tkinter

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Tkinter
- pyttsx3

## How to Run
pip install -r requirements.txt
python main.py
